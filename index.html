Taut Messenger

Full Stack Application

Taut is a full stack clone of the popular messaging service Slack with real time messaging, channels, and full front and back end user authentication. By using WebSockets protocol, Taut achieves real time messages and updates without requiring an explicit HTTP/ajax request from the client. Taut is designed as a Ruby on Rails application with a React app front end using PostgreSQL for its database.

Bitwise Tetris

JavaScript Browser Game

Bitwise Tetris is my own minimalist interpretation of the classic arcade game using CSS3, HTML5, and Canvas. Making use of modern ES6 class syntax, this version of the game has a lightweight but highly modular design that uses bitwise functions to calculate piece rotations and collision detection.
